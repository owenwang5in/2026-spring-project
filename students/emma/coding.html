About My Website ᯓ★

The overall structure and visual design of the website were created by establishing the framework with HTML and styling the layout and color scheme using CSS. Furthermore, I used JavaScript to bring the site to life with interactive features, including designing a "Calculator" page to dynamically handle calculation logic, and utilizing the HTML5 element alongside event listeners to create a "Graffiti" page where users can draw freely. Throughout the development process, from writing code to continuous testing and debugging, I not only became familiar with the complete workflow of web development but also significantly enhanced my logical thinking and problem-solving skills.
