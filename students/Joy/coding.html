About my website

This website was created for my Computer Science class. Throughout the project, I learned how to use HTML, CSS, and JavaScript to design and build web pages. HTML was used to create the structure and content of the website, while CSS was used to style the pages and improve the visual appearance. JavaScript was added to make certain elements interactive.

During the development process, I learned how to create navigation between pages, add images and videos, and use different layout techniques to make the website user-friendly .etc. I also practiced problem-solving skills by debugging code and testing different designs.

Creating this website helped me develop both my coding skills and creativity, and it gave me a better understanding of how websites are built.
